# TRENDEX — JavaScript Event Handling Showcase

TRENDEX is a simple interactive website created to demonstrate JavaScript event handling concepts.  
It shows how different browser events work in real time through user interactions.

## Features

- Custom cursor effect
- Live event status display
- Trending cards with hover effects
- Keyboard input display
- Mouse trail animation
- Search with suggestions
- Scroll progress bar

## Events Used

- Click
- Double Click
- Mouse Over / Hover
- Mouse Move
- Key Down / Key Up
- Input & Focus
- Scroll
- Context Menu

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
TRENDEX/
├── index.html
├── style.css
└── script.js



## How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser
3. Interact with the page to see events in action

## Purpose

This project was built to practice and demonstrate JavaScript DOM event handling for learning purposes.
